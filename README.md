# AJIO Clone

A responsive e-commerce website inspired by AJIO, built using HTML, CSS, and JavaScript.

## Features

* Homepage with Hero Slider
* Men's Fashion Section
* Product Fetching using Fetch API
* Search Functionality
* Price Sorting
* Add to Cart
* Local Storage Integration
* OTP Signup Popup
* Responsive UI

## Technologies Used

* HTML
* CSS
* JavaScript
* Fetch API
* Local Storage
